## 查看已安装的 Forge Release

如果想查看本地已经安装了哪些版本的 Forge Release，可以执行：`forge ls`，输出结果如下所示：

![](./images/forge-ls.png)



## 查看已发布的 Forge Release 列表

Forge 每周都会有新的版本，如何查看我们都发布了哪些版本呢？执行 `forge ls:remote`，即可拿到完整的已经发布过的 Forge 版本

![](./images/forge-ls.png)

类似的，这个列表会把本地已经安装的版本标记出来。
